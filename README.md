# 🏠 HomeFinder Project

## 📌 Overview

This is a property listing web application where users can:

* View properties on a map
* Filter properties by state
* Add / edit property listings
* See property details

---

## 🚀 Tech Stack

* Angular
* TypeScript
* Leaflet (Map)
* RxJS

---

## ⚙️ Setup Instructions

1. Clone the repository

```
git clone <your-repo-link>
```

2. Install dependencies

```
npm install
```

3. Create a `.env` file in the root folder

4. Add required variables (example):

```
API_URL=http://localhost:3000
```

5. Run the project

```
ng serve
```

---

## 📂 Important Note

`.env` file is not included in this repo for security reasons.
Please create it manually using the example above.

---

## ✨ Features

* Split view (Map + Property List)
* Dynamic markers on map
* Hover sync between list and map
* Price formatting (₹, Lakh, Crore)
